Code Standards
~~~~~~~~~~~~~~
The same `code standards`_ adopted by Pulp Smash is adopted by Pulp 2 tests.
This helps to create consistency among the projects.

Review Process
~~~~~~~~~~~~~~
The same `review process`_ adopted by Pulp Smash is adopted by Pulp 2 tests.
